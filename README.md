# generative-models

my playground for generative modeling

## Requirements
The code was developed and tested under the environment:
* OS: Ubuntu 18.04.4 LTS (5.0.0-37-generic)
* CUDA 10.1.243
* PyTorch 1.4.0

## Installation
```shell script
pip install -r requirements.txt
```
